  • OpenStack Essentials
    这本侧重于讲解OpenStack核心服务的基本原理，不到200页，脉络清晰，很快就读完了。
  • OpenStack Cloud Computing Cookbook
    这本侧重于部署，几乎对所有的命令都会有解释，有点罗嗦，不过胜在条理清晰。
